# Windows Server Home Lab (VirtualBox)

## Overview
This project documents the installation and initial configuration of a Windows Server virtual machine using Oracle VirtualBox.  
The lab is designed to practice core system administration tasks such as server installation, network configuration, and server management.

## Lab Environment
- Hypervisor: Oracle VirtualBox
- Guest OS: Windows Server 2019 
- Host OS: Windows 10 
- RAM: 2 GB
- CPU: 1 core
- Storage: 50 GB (VDI)

## Objectives
- Install Windows Server in a virtualized environment
- Configure basic server settings
- Familiarize with Server Manager
- Prepare the server for future roles (AD, DNS, File Server)
